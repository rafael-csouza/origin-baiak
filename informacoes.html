Auto Loot

Primeiramente ligue o sistema, utilizando o seguinte comando:
!autoloot power

Para adicionar os item a lista é bastante fácil, basta falar o seguinte comando:
!autoloot add, [nome do item no SINGULAR]
Exemplos:
!autoloot add, criatura 01
!autoloot add, criatura 02
!autoloot add, criatura 03

Saiba quais são os itens que o autoloot está coletando, para isso, use o seguinte comando:
!autoloot list


Você também pode limpar a lista de coleta a qualquer momento de maneira prática, utilizando o seguinte comando:
!autoloot clear

Se seu autoloot BUGAR use !autoloot desbug

Sistema de Task

Atenção: Todas as task do servidor são necessaria para acessar quests/addons/promotion/etc, essas podem ser conferidos no Quest Log em:
Acessos - Tasks

Para iniciar, receber ou sair uma task, voce deve ir ao npc Pedro Tasks.
Localizado no segundo andar da sala de npc (!tp npc).

Para comecar a task:
hi, task, (NOME DO MONSTRO)

Para receber a task:
hi, receber, yes

Use tambem o comando !task e saiba todas informações sobre suas task.

No servidor existem diversos Globos e Poções de EXP que te ajudam a subir de leveis mais rapido.

Digite !meuinfo ou !jogadorinfo para saber sua taxa de EXP.

  • Poções EXP

    O que é?

    Sistema consiste em ativar as poçoes de EXP e assim ganhar mais EXP por monstros.

    Se seu personagem tiver a pocao de EXP, somente dar USE na pocao conforme imagem abaixo:



    Como funciona?

    Ao ativar a pocao de EXP, esta fica com efeito somente enquanto seu personagem estiver online.

    Caso seu personagem estiver RR menor que o necessario para usar a pocao, esta automaticamente sera ativada.



    Voce consegue obter as poções RR menores que o necessario para o uso, bom para ir acumulando para quando necessitar o uso.

  • Globos EXP

    O que é?

    Sistema consiste em desbloquear os globos de EXP e assim ganhar mais EXP por monstros.

    Se seu personagem tiver todos os requisitos, somente dar USE no globo conforme imagem abaixo:



    Como funciona?

    Ao desbloquear o globo de EXP, este continua para sempre no seu personagem.

  • Atenção: Todos globos e poções do servidor podem ser conferidos no Quest Log em:

    EXP - Globos
    EXP - Pocoes

Comandos

Alguns comandos úteis:

!meuinfo - Mostra seu level / resete / mana / vida
!magias - Mostra suas magias e RR necessários
!tp - Mostra a lista de lugares que pode-se teleportar
!autoloot - Usa o autoloot e mostra informações de como usar
!caixa - Ve seu caixa no banco
!resetar - Reseta seu personagem se o mesmo possui level necessário
!msgtodos - Manda uma menssagem para todos do servidor
!mochila - Compra uma mochila com seu nome
!bless - Compra todas bless, menos a bless PVP
!jogadorinfo - Mostra informações de jogadores
!jogadoritem - Mostra itens de jogadores
!task - Mostra suas task completas e o progresso da task atual
